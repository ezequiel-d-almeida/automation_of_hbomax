# HBO Desktop Automation

Projeto de automação desktop desenvolvido em Python para automatizar ações repetitivas na navegação e reprodução da HBO.

## Tecnologias
- Python
- PyAutoGUI
- Programação Orientada a Objetos
- Modularização

## Conceitos aplicados
- Encapsulamento
- Separação de responsabilidades
- Reutilização de código
- Estrutura em classes

## Estrutura
```bash
src/
classes.py
main.py
```

## Objetivos
Reduzir tarefas manuais repetitivas no controle de mídia.

## Aprendizados
Este projeto marcou minha evolução de scripts lineares para uma arquitetura orientada a objetos, com foco em escalabilidade e manutenção.
